# aiiinotate

Lightweight and fast annotation server for IIIF and Mirador 3

---

## Install

```bash
bash setup.sh
```

--- 

## Usage

Start the app in dev:

```bash
npm start
```

Start the app in prod:

```bash
npm prod
```

Test the app:

```bash
npm test
```

Run the CLI:

```bash
npm cli
```

Process migrations:

```
npm migrate-(init|make|apply|revert|revert_all)
```
